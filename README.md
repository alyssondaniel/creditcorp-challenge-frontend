# Credicorp Challenge

Prerequisites

You will need installed on your system:
  - Node version 14.16 or greater.
  - Yarn version 1.22.10 or greater.
  - Backend started


## Setup
```
yarn install
```
## Run development

```
  yarn dev
```

## Run production

```
  yarn build
  yarn start
```

## How to use

Access [`http://localhost:3000`](http://localhost:3000)
